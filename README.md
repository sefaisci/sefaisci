# Hi, I'm Sefa İşci

Agentic AI / LLM Engineer focused on building reliable AI agents, RAG-memory systems, guardrails, and document intelligence workflows.

I work on production-oriented AI systems where reasoning quality, structured outputs, retrieval behavior, and failure handling matter as much as model choice.

## What I Build

- **Agentic AI systems**: supervisor-worker agents, plan-act-reflect workflows, tool-using agents, and multi-agent orchestration patterns.
- **Reliable LLM engineering**: guardrails, schema validation, JSON/Pydantic outputs, evaluation harnesses, prompt-injection defense, and audit-ready traces.
- **RAG and memory systems**: retrieval pipelines, vector search, long-term agent memory, semantic search, and context engineering.
- **Document intelligence**: OCR, PDF ingestion, structured extraction, validation workflows, and decision-support pipelines.
- **Applied ML systems**: NLP, computer vision, anomaly detection, recommendation systems, and similarity search.

- Treat retrieval quality as an engineering problem, not a checkbox.
- Design agents with observable failure modes.
- Keep demos reproducible, synthetic, and free of proprietary data.
